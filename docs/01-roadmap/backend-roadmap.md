1. Project Setup
2. Product Management

6. Reporting

Goal:

- CRUD product API

- validation
- service layer

## Authentication

Goal:

- register user
- login user
- JWT token
- protected endpoints

## Sales

Goal:

- create sale
- list sales
- calculate total amount
- connect sale item with product

## Reporting

Goal:

- simple sales summary
- product count
- revenue summary
