# Glossary

## UMKM

Usaha Mikro, Kecil, dan Menengah.

## Product

Barang atau jasa yang dijual oleh UMKM.

## Sale

Transaksi penjualan product kepada customer.

## Admin

User yang mengelola data product, sales, dan account.

## DTO

Data Transfer Object. Object khusus untuk request dan response API.

## Entity

Object yang merepresentasikan table database melalui JPA.

## Repository

Layer yang bertanggung jawab mengakses database.

## Service

Layer yang menyimpan business logic.

## Controller

Layer HTTP yang menerima request dan mengembalikan response.
