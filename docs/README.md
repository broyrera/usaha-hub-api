# Documentation Index

Repository ini memakai dokumentasi markdown sebagai project management system.

## Folder

- `00-project` - informasi dasar produk dan scope
- `01-roadmap` - arah belajar dan roadmap backend
- `02-sprints` - sprint berjalan dan backlog dekat
- `03-architecture` - gambaran sistem dan design teknis
- `04-learning-notes` - catatan belajar konsep Spring Boot
- `05-decisions` - Architecture Decision Records
- `06-api` - kontrak API per module
- `07-review` - review, refactor, dan mistakes log

## Rule

Dokumentasikan saat membantu belajar atau maintenance.

Tidak semua hal perlu ditulis sebelum coding.
